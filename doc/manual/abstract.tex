\begin{abstract}
Jahob is a system for verifying data structure consistency
of programs written in a subset of Java.  This manual
describes the subset of Java accepted by Jahob, as well as
the use of Jahob specification constructs such as
invariants, specification variables, and procedure
contracts.
\end{abstract}
